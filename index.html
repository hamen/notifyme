NotifyMe - SamePlace Power Pack

Description

NotifyMe is a Mozilla Firefox / SamePlace extension. It adds some cool features to SamePlace. Currently it's available for Mozilla Firefox. Thunderbird support has been removed in last release.

Main features

  • Automatic reconnection after a disconnection
  • Status message and status, i.e. Available, Away or Busy, saving and restoring
  • Notification popup on incoming messages from users and/or rooms with Firefox minimized, not on focus or on another desktop
  • Sound on notifications
  • "Watch on user": get notified on somebody status changes
  • Automatic favorites rooms connection at login
